## Python

### Dépendances

- [Interpréteur Python](https://www.python.org)
- [Git](https://git-scm.com/)
- [Visual Studio Code](https://code.visualstudio.com/) ou autre IDE (éditeur de texte simple déconseillé)
- [Extension _Microsoft Python_ pour VS Code](https://marketplace.visualstudio.com/items?itemName=ms-python.python)
- [Extension _Jupyter_ pour VS Code](https://marketplace.visualstudio.com/items?itemName=ms-toolsai.jupyter)

### Cloner ce dépôt

```bash
git clone https://adressedecedepot.git
```

Puis ouvrir tout le dossier cloné dans l'IDE (et non pas 1 seul fichier).
